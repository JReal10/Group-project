# Piecewise Linear Functions

Each section of a function is defined by providing a range and the function for the specified range.
$$
f(x) = \left\{
    \begin{array}{ll}
        x^2 & if \, x<2\\
        6 & if \, x=2\\
        10-x & if \; x>2 \; and \; x\leq6
    \end{array}
\right.
$$
